# OSDU Admin

This project aims to offer edit and create capabilities for OSDU records.

## How to spin it up

Its the usual nodejs project innit.
First clone it, then `npm i` it, then `npm run dev` it

## Version 1.0.0 Roadmap

- Provide a kind, fetch the record and populate a HTML form.
- Edit and save a record.
- Create a new record.
- Tracking changes in IndexedDB. Undo changes, undo all changes.
- Tabbed interface, open several records.
